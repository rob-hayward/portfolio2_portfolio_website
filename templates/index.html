Portfolio Project 1 - The Morse Machine

A text based/command line project using Python and PyCharm to write a program that takes any String input and converts it into Morse Code.

Although this is a relatively simple piece of functionality to program, it is a nice little project to demonstrate a variety of programming skills:

  • Reading from CSV files using Pandas.
  • Converting Pandas data frames into more usable dictionary formats.
  • Working with lists and dictionaries to collect and output desired data.
  • Using while loops to allow the user to continue or terminate program.
  • Catching common exceptions to enhance user experience and prevent crashes.
Even on a small project such as this, it is important for me to not only write good functioning code but also to practice creating a good user experience and ensure that I use best coding practices to make my code easy to read and work with for myself or others in the future.

Portfolio Project 3 - Tic Tac Toe Vs Robo

A text based/command line project using Python and PyCharm to program a 1 or 2 player version of the classic game tic-tac-toe.

This was a fun project that I could feel myself digging into and wanting to do well on. The task can be accomplished in many ways, but it felt important for me not only to create a functional game with a good user experience, but to also use methods and logic that would be scalable to larger tasks. For this reason I decided to use numpy to structure and check my data. I also created a simple A.I. to play against. I'm pleased with the end result.

  • Leveraging numpy to practice using large scale concepts at a small scale.
  • Thinking through logical orders of operation and constructing elegant and clear code.
In this project I've continued to try and create clean and clear solutions to problems. These take slightly longer to write, however, I feel that it is excellent practice in creating strong foundations in my code that can be understood and built on by future project contributors.
